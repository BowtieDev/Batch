# Batch
The batch folder for CSC1A

# How do I install it?

Download the available zip folder and extract it

# How does it work?

1. Drag the folder named "Batch" into your project repository (the folder with bin, docs, and src)
2. To start, run start.bat
3. Now all the necessary files will be moved to where they will be used
4. In src there will be compile.bat, del.bat, and run.bat
5. Use compile.bat to make an exe for testing
6. Use run.bat to run the testing exe
7. Use del.bat to get rid of it.
8. When the whole project is done, run final_compile.bat
9. Now all batch files will be back in Batch, and everything should be set up for submission
10. Move the Batch folder out of the folder before submitting
